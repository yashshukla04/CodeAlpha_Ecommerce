### CodeAlpha Internship - Task 1

A full-stack e-commerce application built with **HTML, CSS, JavaScript (Frontend)** and **Node.js, Express, MongoDB (Backend)**. This project features user authentication, dynamic product listings, a shopping cart, and a complete order processing system.

##  Features

* **User Authentication:** Secure User Registration and Login system (stored in MongoDB).
* **Dynamic Product Listing:** Fetches products directly from the database with real-time pricing (INR).
* **Shopping Cart:** Add/Remove items, view total price, and manage cart state.
* **Order Processing:** Checkout functionality that saves orders to the database for the logged-in user.
* **Product Details:** "Quick View" modal to see product descriptions and details without leaving the page.
* **Responsive Design:** Fully mobile-responsive UI using Bootstrap 5.
* **Database Seeder:** One-click API route to populate the database with dummy products.

##  Tech Stack

* **Frontend:** HTML5, CSS3, Bootstrap 5, Vanilla JavaScript (Fetch API)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (Cloud NoSQL)
* **ODM:** Mongoose
